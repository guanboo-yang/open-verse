# 綱目缺漏記錄(舊站有、新站無)

比對舊站(recoveryversion.com.tw)與新站(line.twgbr.org)綱目,記錄舊版有但新版
找不到對應的條目。比對條件:同章、節差 ≤2、標題正規化後相似度 >0.55。
共 40 條:續 25、非續 15。

## 決定(2026-06-15)

**A 類(續,25 條)→ 全補。** 這些是父段(L2/L3)的「跨章 context 重述」。新綱目其實
有完整階層(子項 L4 全在),只是不重複父段標題。理論上應全補,讓讀者捲到跨章段落
中段時仍知道身處哪個大段。實作方向:寫 merge 腳本,把舊綱目的「續」父段對齊到新綱目
對應章節插入(每條的插入點見 scripts/output/gap_continued.txt 與下方規律)。

**規律**:每條「續」應插在「新綱目該章該節的 L4 子項之前」,層級沿用舊的父段層級。
唯一特例:創世記 27:27 新綱目 v±1 完全無條目(讀者會迷路),最該補。

**B 類(非續,15 條)**:多為改寫/錨點位移(內容仍在);真缺僅列王紀上 10:26~28
「購建戰車和馬兵作為防衛」,一併補回。

> 狀態:已記錄,實作待後續。

## A. 續延續標記(25)— 新站省略的「跨章段落 context 重述」

舊站在一段綱目跨越多章時,於每章頂重述「…－續」,提醒讀者仍在該段落。新站幾乎
不重述,故缺漏。**這些對閱讀的段落定位很有幫助,建議補回。**
